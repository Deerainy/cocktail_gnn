# 前端对接说明 - 中英文字段

## 概述

后端API已支持中英文字段返回，具体字段如下：

## 1. 配方信息接口

### 接口地址
- `GET /api/recipes` - 获取所有配方
- `GET /api/recipes/{recipe_id}` - 获取单个配方详情

### 中英文字段

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `name` | 配方英文名 | "Flor de Amaras" |
| `recipe_name_zh` | 配方中文名 | "阿马罗之花" |

### 返回示例
```json
{
  "code": 0,
  "message": "ok",
  "data": {
    "recipe_id": "1",
    "name": "Flor de Amaras",
    "recipe_name_zh": "阿马罗之花",
    "instructions": "...",
    "glass": "coupe",
    "tags": "tequila, mezcal, cocktail",
    "image_url": "https://example.com/image.jpg",
    "is_alcoholic": true
  }
}
```

## 2. 原料信息接口

### 接口地址
- `GET /api/recipes/{recipe_id}/ingredients` - 获取配方原料列表

### 中英文字段

| 字段名 | 说明 | 示例 |
|--------|------|------|
| `name_norm` | 原料原始名称 | "mezcal" |
| `canonical_name` | 原料标准英文名 | "mezcal" |
| `canonical_name_zh` | 原料标准中文名 | "梅斯卡尔酒" |
| `anchor_name` | 风味锚点名称 | "agave" |

### 返回示例
```json
{
  "code": 0,
  "message": "ok",
  "data": [
    {
      "ingredient_id": "1",
      "line_no": 1,
      "amount": 1.5,
      "unit": "oz",
      "role": "base_spirit",
      "raw_text": "1.5 oz Mezcal",
      "ingredient": {
        "ingredient_id": "1",
        "name_norm": "mezcal",
        "canonical_id": "1",
        "canonical_name": "mezcal",
        "canonical_name_zh": "梅斯卡尔酒",
        "category": "Spirit",
        "abv": 40.0,
        "anchor_name": "agave",
        "anchor_form": "spirit"
      }
    }
  ]
}
```

## 3. 前端使用建议

### 显示优先级
1. **配方名称**：优先显示 `recipe_name_zh`，为空时显示 `name`
2. **原料名称**：优先显示 `canonical_name_zh`，为空时显示 `canonical_name`

### 代码示例
```javascript
// 获取配方列表
fetch('http://127.0.0.1:8000/api/recipes')
  .then(response => response.json())
  .then(result => {
    if (result.code === 0) {
      const recipes = result.data.map(recipe => ({
        id: recipe.recipe_id,
        // 优先显示中文名，没有则显示英文名
        name: recipe.recipe_name_zh || recipe.name,
        originalName: recipe.name, // 保留英文名备用
        image: recipe.image_url,
        isAlcoholic: recipe.is_alcoholic
      }));
      // 渲染配方列表...
    }
  });

// 获取原料列表
fetch('http://127.0.0.1:8000/api/recipes/1/ingredients')
  .then(response => response.json())
  .then(result => {
    if (result.code === 0) {
      const ingredients = result.data.map(item => ({
        id: item.ingredient_id,
        amount: item.amount,
        unit: item.unit,
        role: item.role,
        // 优先显示中文名
        name: item.ingredient.canonical_name_zh || item.ingredient.canonical_name,
        englishName: item.ingredient.canonical_name
      }));
      // 渲染原料列表...
    }
  });
```

## 4. 注意事项

1. **字段可能为空**：`recipe_name_zh` 和 `canonical_name_zh` 可能为 `null`，请做好空值处理
2. **编码问题**：数据库已使用 `utf8mb4` 编码，支持完整的Unicode字符（包括emoji）
3. **API基础地址**：`http://127.0.0.1:8000/`
